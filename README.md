
# ⚙️  My Dotfiles

## Installation

### Clone
```
cd ~
git clone https://github.com/andtyl/dotfiles
```

### Oh My Zsh
`curl -L https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh | sh`

### Git Submodules
`git submodule update --init`

### Symlinks
```
ln -s dotfiles/.gitconfig
ln -s dotfiles/.gitignore_global
ln -s dotfiles/.vim
ln -s dotfiles/.vimrc
ln -s dotfiles/.zshrc
```

✅ Done
