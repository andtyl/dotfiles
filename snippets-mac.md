## Keystrokes

`shift + alt + 7` - print '\'
`alt + 7` - print '|'
`shift + option` - maximize window
`ctrl + command + f` - fullscreen window
`ctrl + ←/→` - switch desktop

## Tweaks

### Window maximize animation

disable: `defaults write -g NSWindowResizeTime -float 0.003`
enable: `defaults delete -g NSWindowResizeTimes`
